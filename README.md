# .github

This repository contains the [GitHub Community Health Files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for the **OrthyxAI** organization.

## Files

- `profile/README.md` — Organization profile displayed on GitHub
- `CODE_OF_CONDUCT.md` — Community standards
- `CONTRIBUTING.md` — How to contribute
- `ISSUE_TEMPLATE/` — Issue templates
- `pull_request_template.md` — PR template

These files will be used as defaults for all repositories in this organization that don't have their own versions.
